# Unit 3: Advanced Functions & Algorithms - Course Package

## Package Contents

This comprehensive course package includes everything needed to teach a 27-day high school Python unit on advanced functions and algorithms.

### Files Included

**Main Documents:**
- `Unit 3 - Advanced Functions & Algorithms.md` - Complete hyperbook with all content
- `ReferenceGuide.md` - Quick reference for all syntax and concepts
- `UnitPractice.md` - 20 practice problems (8 beginner, 8 intermediate, 4 challenge)
- `UnitAssessment.md` - Complete unit assessment (multiple choice, short answer, coding)

**Daily Notes (27 files):**
- `Daily Notes/Day1.md` through `Daily Notes/Day27.md`
- Each day includes:
  - Learning target
  - Key concepts
  - 10+ code examples
  - Guided practice activity
  - 3 practice problems (beginner, intermediate, challenge)
  - Complete solutions

### Curriculum Overview

**Days 1-5:** Functional Programming Fundamentals
- Lambda functions
- map(), filter(), zip(), enumerate()
- Functional programming patterns

**Days 6-9:** Comprehensions & Generators
- List, dict, set comprehensions
- Nested comprehensions
- Generator expressions and yield

**Days 10-14:** Recursion
- Base and recursive cases
- Classic problems (factorial, Fibonacci)
- Complex recursion (Towers of Hanoi, flattening)
- Recursion vs. iteration

**Days 15-17:** Algorithms & Complexity
- Algorithm design and pseudocode
- Big O notation (time complexity)
- Space complexity analysis

**Days 18-24:** Advanced Functions
- Closures and scope (LEGB rule)
- Decorators
- Practical decorators (timing, logging, memoization)
- Iterators and custom iterators
- Generators deep dive

**Days 25-27:** Review & Assessment
- Review day with integration challenges
- Practice day with algorithm challenges
- Unit assessment

### Learning Outcomes

Upon completing this unit, students will be able to:

1. Write and use lambda functions for simple operations
2. Apply map(), filter(), and zip() to transform data
3. Create comprehensions (list, dict, set, generator)
4. Implement recursive solutions with base and recursive cases
5. Analyze algorithms using Big O notation
6. Write decorators to modify function behavior
7. Create closures that manage state
8. Build custom iterators and generators
9. Choose appropriate tools for different problems
10. Optimize code for readability and performance
