# Unit Practice Bank

1. **Beginner:** MDA Analysis — Pick a game you know and analyze it through the MDA framework (mechanics, dynamics, aesthetics).

2. **Beginner:** Canvas Exploration — Create a Canvas with 5+ different UI elements (buttons, sliders, text, images). Adjust anchors and test on different screen resolutions.

3. **Beginner:** TextMeshPro Formatting — Create text with rich text tags: bold, colored, sized, and styled. Test different fonts.

4. **Beginner:** Button Network — Create a menu with buttons that link to different panels/scenes. Implement proper transitions.

5. **Intermediate:** Score System — Implement a complete score system with display, save/load, and high score tracking.

6. **Intermediate:** Health Bar UI — Create both a number display (100/100) and a visual bar that empties as health decreases.

7. **Intermediate:** Pause Menu — Create a pause menu that freezes gameplay (Time.timeScale = 0) and unfreezes on resume.

8. **Intermediate:** PlayerPrefs Practice — Save 5+ different player settings and verify they persist across scenes.

9. **Intermediate:** Scene Transitions — Create smooth fade-in/fade-out transitions between scenes using CanvasGroup alpha.

10. **Challenge:** Level Design Document — Write a one-page level design document for a platformer level, including difficulty progression.

11. **Challenge:** Tutorial Design — Design an onboarding level that teaches one new mechanic at a time without text instructions.

12. **Challenge:** Difficulty Scaling — Create a difficulty selection menu that changes gameplay parameters (enemy speed, time limit, etc.).

13. **Challenge:** Leaderboard Simulation — Create a UI that displays top 5 scores, with current player highlighted.

14. **Challenge:** Responsive UI — Design a UI that works correctly on mobile (portrait), tablet (landscape), and desktop resolutions.

15. **Challenge:** Game Loop Analysis — Create a flowchart describing the core loop of your favorite game and identify feedback, input, resolution, and repetition.
