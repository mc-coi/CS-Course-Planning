# Capstone Project Practice Tasks

## Comprehensive Capstone Tasks

1. **Task 1 — Beginner:** Write a complete Game Design Document (2-3 pages) for your capstone game.

2. **Task 2 — Beginner:** Design and implement the MVP (Minimum Viable Product) for your game.

3. **Task 3 — Intermediate:** Implement all major systems (Player, Enemy AI, Health, Scoring, UI) and integrate them.

4. **Task 4 — Intermediate:** Playtest with 3-5 players and document feedback.

5. **Task 5 — Intermediate:** Iterate based on playtesting feedback.

6. **Task 6 — Challenge:** Add polish (particles, sounds, screen shake, animations) to enhance game feel.

7. **Task 7 — Challenge:** Prepare build settings and export to WebGL.

8. **Task 8 — Challenge:** Create presentation and reflection document (1-2 pages).
