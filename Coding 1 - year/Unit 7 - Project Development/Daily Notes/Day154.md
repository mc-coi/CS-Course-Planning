# Day 154: Building a Working Prototype (MVP)

**Learning Target:** I can build a Minimum Viable Product that demonstrates core functionality, even if it's not polished.

---

## Key Concepts

### What is an MVP (Minimum Viable Product)?

An **MVP** is your program with only the **essential core features**. No nice-to-haves. No polish. Just the basic, working version.

**Why build an MVP?**
- Get working code quickly (confidence boost)
- Find design problems early (before coding everything)
- Have something to test and refine
- Prove the concept works

**MVP Criteria:**
- ✅ Core features work
- ✅ No crashes on normal input
- ✅ Data flows correctly between functions
- ✗ Code doesn't need to be beautiful
- ✗ Error handling can be basic
- ✗ Documentation can be minimal

### Bottom-Up Coding Strategy (Reminder)

Build and test the **simplest function first**, then build on top of it:

1. Code `calculate_average()` → test alone
2. Code `assign_grade()` → test alone
3. Code `get_scores()` → test with #1 and #2
4. Code `display_report()` → integrate all
5. Code `main()` → orchestrate everything

## Code Examples

### Example: Grade Tracker MVP (Skeleton)

```python
"""
Grade Tracker - MVP (Minimum Viable Product)
Day 154: Core functions working, integrated, ready for testing.

Core features:
1. Enter student name ✓
2. Enter scores with validation ✓
3. Calculate average ✓
4. Assign letter grade ✓
5. Display report ✓
"""

# ════════════════════════════════════════════════════
# HELPER FUNCTIONS (start with these)
# ════════════════════════════════════════════════════

def calculate_average(scores):
    """Return mean of scores (or 0.0 if empty)."""
    if not scores:
        return 0.0
    return sum(scores) / len(scores)

def assign_grade(average):
    """Convert average to letter grade."""
    if average >= 90: return "A"
    if average >= 80: return "B"
    if average >= 70: return "C"
    if average >= 60: return "D"
    return "F"

# ════════════════════════════════════════════════════
# INPUT FUNCTIONS
# ════════════════════════════════════════════════════

def get_student_name():
    """Get and validate student name."""
    name = input("Student name: ").strip()
    while not name:
        print("Name cannot be empty.")
        name = input("Student name: ").strip()
    return name

def get_scores():
    """Collect scores with validation."""
    scores = []
    print("Enter scores one at a time. Type 'done' when finished.")
    while True:
        entry = input(f"Score {len(scores) + 1} (or 'done'): ").strip()
        if entry.lower() == "done":
            if not scores:
                print("You must enter at least one score.")
                continue
            break
        try:
            score = float(entry)
            if 0 <= score <= 100:
                scores.append(score)
            else:
                print("Score must be between 0 and 100.")
        except ValueError:
            print("Please enter a valid number.")
    return scores

# ════════════════════════════════════════════════════
# OUTPUT FUNCTIONS
# ════════════════════════════════════════════════════

def display_report(name, scores, average, grade):
    """Print a formatted report."""
    border = "=" * 40
    print(f"\n{border}")
    print(f"  Grade Report: {name}")
    print(border)
    print(f"  Scores:   {[round(s, 1) for s in scores]}")
    print(f"  Average:  {average:.1f}")
    print(f"  Grade:    {grade}")
    status = "PASSING" if average >= 70 else "FAILING"
    print(f"  Status:   {status}")
    print(border)

# ════════════════════════════════════════════════════
# MAIN
# ════════════════════════════════════════════════════

def main():
    """Run the Grade Tracker MVP."""
    print("=" * 40)
    print("   Welcome to Grade Tracker")
    print("=" * 40 + "\n")

    # Get data
    name = get_student_name()
    scores = get_scores()

    # Calculate
    average = calculate_average(scores)
    grade = assign_grade(average)

    # Display
    display_report(name, scores, average, grade)

    print("\nThank you for using Grade Tracker!")

if __name__ == "__main__":
    main()
```

## Guided Practice

### Build Your MVP (Full Period)

**Step 1: Set Up Your Project File (5 min)**
```python
"""
[Project Name] - MVP
Minimum Viable Product with core features working.
"""

# Your code here
```

**Step 2: Code Foundational Functions (20 min)**
- Identify 2–3 functions that don't depend on others
- Code them
- Test them individually (write small test code below them)
- Don't move on until they work

**Step 3: Code Input/Processing Functions (15 min)**
- Add the next layer
- Test each one
- Make sure data types match (e.g., if one returns list, next accepts list)

**Step 4: Code Main (5 min)**
- Connect everything
- Call functions in order
- Run the entire program once

**Step 5: Catch and Fix Bugs (10 min)**
- Run your MVP with normal input
- Make note of any crashes or odd behavior
- Fix them (don't worry about perfect code yet)
