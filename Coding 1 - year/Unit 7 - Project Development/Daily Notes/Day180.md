## Key Concepts

### Portfolio Review

A **portfolio** is a collection of your best work. It's useful for:
- College applications
- Job interviews
- Demonstrating skills
- Building your reputation

**What to include:**
- Your capstone project (with code and documentation)
- 2-3 other projects you're proud of
- Links to your code (GitHub)
- Brief descriptions of each project
