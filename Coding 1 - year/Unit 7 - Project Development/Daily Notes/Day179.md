## Key Concepts

### CS Career Paths

After Coding I, you have many directions:

**Web Development**
- Frontend: HTML, CSS, JavaScript, React
- Backend: Python (Django, Flask), Node.js, Java
- Fullstack: Both frontend and backend

**Mobile App Development**
- iOS (Swift)
- Android (Java, Kotlin)
- Cross-platform (React Native, Flutter)

**Data Science & Machine Learning**
- Python, pandas, NumPy, scikit-learn
- Statistical analysis
- Building predictive models

**Game Development**
- Unity (C#)
- Unreal Engine (C++)
- Godot (GDScript)

**Cybersecurity**
- Network security
- Ethical hacking
- System administration

**Software Engineering**
- Large-scale application development
- System design
- Team collaboration

**AI & Automation**
- Machine learning
- Natural language processing
- Robotics

### What's Next: Coding II

Coding II builds on Coding I:

**Topics:**
- Object-Oriented Programming (classes, inheritance)
- More advanced data structures (linked lists, trees, graphs)
- Algorithms (sorting, searching, optimization)
- Project management and collaboration
- Advanced debugging and testing

**Skills:**
- Think in terms of "objects" and "classes"
- Design complex systems
- Analyze algorithm efficiency
- Work with real-world codebases

## Coding I → Coding II: The Jump

**Coding I:** Learning the basics (variables, loops, functions)
**Coding II:** Learning to think like a programmer (OOP, design patterns, algorithms)

**Simple metaphor:**
- Coding I: Learning to write sentences
- Coding II: Learning to write novels
