# Day 173: Building Your Presentation

**Learning Target:** I can create a clear, well-organized presentation that demonstrates my project effectively.

---

## Key Concepts

### Presentation Structure (5-7 minutes)

1. **Introduction** (30 sec): What is your project? What problem does it solve?
2. **Demo** (2-3 min): Show it working with normal and edge case input
3. **Code Walkthrough** (1-2 min): Explain 1-2 key functions
4. **Challenges & Solutions** (45 sec): What was hard? How did you solve it?
5. **Key Learnings** (30 sec): What did you learn?
6. **Questions** (1 min): Invite audience questions

### Presentation Medium

**Option 1: Live Demo Only** (Simplest)
- No slides, just run program live
- Risky if something crashes

**Option 2: Slides + Demo** (Recommended)
- Slides for structure and info
- Live demo of working program
- Safety: if demo fails, fall back to slides

---

## Slide Template (6-8 slides total)

**Slide 1: Title**
- Project name
- Your name
- Date

**Slide 2: Problem**
- What problem does it solve?
- Who uses it?

**Slide 3: Solution**
- Your program in a nutshell
- Key features

**Slide 4: Demo Setup**
- "Now I'll show you how it works"
- What you're about to show

**Slide 5: Code Walkthrough**
- Key function name
- Important logic (text, not code)

**Slide 6: Challenges**
- "The hardest part was..."
- How you solved it

**Slide 7: Learnings**
- What you learned
- How you grew as a programmer

**Slide 8: Thank You**
- "Questions?"

---

## Guided Practice

### Build Your Presentation (Full Period)

**Step 1: Outline Content (10 min)**
- Write what you'll say on each slide
- Time yourself (aim for 5-7 minutes)

**Step 2: Create Slides (20 min)**
- Open PowerPoint, Google Slides, or Canva
- Add slides following template
- Keep text minimal (not wall of words)
- Add visuals if helpful

**Step 3: Prepare Demo (10 min)**
- Decide what you'll demo
- Write down test inputs
- Practice running demo (make sure it works!)

**Step 4: Review & Refine (10 min)**
- Read slides aloud
- Adjust timing
- Check for clarity

**Step 5: Save & Backup (5 min)**
- Save your slides
- Save a backup copy
- Bring to class on USB drive
