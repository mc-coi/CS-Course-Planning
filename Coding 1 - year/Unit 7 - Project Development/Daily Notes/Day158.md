
**Check at each step:**

## Testing Strategy for Work Days

### Unit Testing Checklist

For each function you write:

```python
def calculate_average(scores):
    """Calculate mean of scores (or 0.0 if empty)."""
    if not scores:
        return 0.0
    return sum(scores) / len(scores)

# TEST IT:
print("Testing calculate_average:")
assert calculate_average([85, 92, 78]) == 85.0, "Normal case failed"
assert calculate_average([100]) == 100.0, "Single score failed"
assert calculate_average([]) == 0.0, "Empty list failed"
assert calculate_average([0, 100]) == 50.0, "Extremes failed"
print("✓ All tests passed!")
```

### Integration Testing

After integrating functions:

```python
# Test the pipeline: input → calculate → display
name = get_student_name()  # Test with "Alice"
scores = get_scores()  # Test with [85, 92, 78]
avg = calculate_average(scores)  # Should get 85.0
grade = assign_grade(avg)  # Should get "B"
display_report(name, scores, avg, grade)  # Should print nicely
```
