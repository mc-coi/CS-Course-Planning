## Key Concepts

### Debugging Workflow

1. **Reproduce the bug** — Find exact steps to trigger it
2. **Isolate it** — Which function is failing?
3. **Diagnose it** — Why is it failing? (Use print statements)
4. **Fix it** — Make minimum change needed
5. **Verify the fix** — Re-run the test case
6. **Check for side effects** — Did fix break anything else?

### Print Debugging Technique

```python
def calculate_average(scores):
    print(f"[DEBUG] Received scores: {scores}")
    print(f"[DEBUG] Type of scores: {type(scores)}")
    if not scores:
        return 0.0
    total = sum(scores)
    print(f"[DEBUG] Total: {total}")
    avg = total / len(scores)
    print(f"[DEBUG] Average: {avg}")
    return avg

# Test:
result = calculate_average([85, 92, 78])
# Output shows each step
```
