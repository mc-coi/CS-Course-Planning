# Day 164: File I/O Integration (Save/Load Data)

**Learning Target:** I can implement file save and load functionality to persist data between program runs.

---

## Key Concepts

### Why File Persistence?

Without it: Every time program closes, data is lost.
With it: Data survives between program runs.

### File Format Design

**Option 1: Plain Text (Simple)**
```
Alice
85 92 78 88
85.0
B
```

**Option 2: CSV (Spreadsheet)**
```
name,scores,average,grade
Alice,"85,92,78,88",85.0,B
```

**Option 3: JSON (Structured)**
```json
{
  "students": [
    {
      "name": "Alice",
      "scores": [85, 92, 78, 88],
      "average": 85.0,
      "grade": "B"
    }
  ]
}
```

For this course, stick with **plain text or CSV**.

### Read/Write Patterns

**Write to file:**
```python
def save_data(filename, students):
    """Save student data to file."""
    try:
        with open(filename, "w") as f:
            for name, data in students.items():
                scores_str = ",".join(str(s) for s in data["scores"])
                f.write(f"{name}|{scores_str}|{data['average']:.1f}|{data['grade']}\n")
        print(f"✓ Data saved to {filename}")
        return True
    except IOError as e:
        print(f"✗ Error saving: {e}")
        return False
```

**Read from file:**
```python
def load_data(filename):
    """Load student data from file."""
    students = {}
    try:
        with open(filename, "r") as f:
            for line in f:
                line = line.strip()
                if not line:
                    continue
                parts = line.split("|")
                if len(parts) == 4:
                    name, scores_str, avg_str, grade = parts
                    scores = [float(s) for s in scores_str.split(",")]
                    students[name] = {
                        "scores": scores,
                        "average": float(avg_str),
                        "grade": grade
                    }
        print(f"✓ Loaded {len(students)} students from {filename}")
        return students
    except FileNotFoundError:
        print(f"File {filename} not found. Starting with empty data.")
        return {}
    except Exception as e:
        print(f"✗ Error loading file: {e}")
        return {}
```

### Implement File I/O (Full Period)

**Step 1: Design File Format (5 min)**
- How will you store data?
- One student per line? One file per student?
- What delimiters (commas, pipes, tabs)?

**Step 2: Write `save_data()` (15 min)**
- Open file for writing
- Loop through your data
- Write each record
- Handle errors

**Step 3: Test Save (5 min)**
- Run program
- Save data
- Check file exists and looks right

**Step 4: Write `load_data()` (15 min)**
- Open file for reading
- Parse each line
- Reconstruct data structure
- Handle missing file

**Step 5: Test Load (10 min)**
- Restart program
- Load saved data
- Verify it matches what you saved

**Step 6: Integrate (5 min)**
- Add save/load to your main menu
- Test full cycle: save → close → load
