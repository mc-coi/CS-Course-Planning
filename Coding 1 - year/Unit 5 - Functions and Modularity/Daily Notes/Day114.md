### Testing Checklist

For each major feature of your program, test:
- **Normal case** — Input that should work normally
- **Edge cases** — Empty inputs, extreme values, minimum/maximum
- **Error cases** — Invalid inputs (if your program should handle them)

**Example for Quiz Game:**
- Normal: Answer a question correctly
- Normal: Answer a question incorrectly
- Edge: Quiz with only 1 question
- Edge: Quiz with 10 questions
- Error: User enters blank answer
- Error: User enters answer in different case (should still work)

### Common Bugs to Look For

1. **Off-by-one errors** — Loops processing one too many or one too few items
2. **Type mismatches** — Comparing strings to ints, or forgetting to convert input
3. **Empty list handling** — Division by zero when calculating average of empty list
4. **Index out of range** — Accessing `list[5]` when list only has 3 items
5. **Variable scope** — Using a variable that hasn't been defined

### Debugging Strategy

When something doesn't work:
1. **Read the error message** — Python tells you what's wrong and where
2. **Isolate the problem** — Run just the broken function, not the whole program
3. **Use print statements** — Print variables to see their values
4. **Check assumptions** — Make sure inputs are what you think they are
5. **Test smaller pieces** — Test helper functions before the whole program

### Example: Testing a Grade Tracker

```python
# Test data
grades_alice = [85, 90, 88, 92]
grades_empty = []
grades_single = [95]

# Test: calculate_average with normal data
avg = calculate_average(grades_alice)
assert avg == 88.75, f"Expected 88.75, got {avg}"
print("✓ Average calculation works")

# Test: calculate_average with empty list
avg_empty = calculate_average(grades_empty)
assert avg_empty == 0, f"Expected 0 for empty, got {avg_empty}"
print("✓ Empty list handling works")

# Test: calculate_average with single grade
avg_single = calculate_average(grades_single)
assert avg_single == 95, f"Expected 95, got {avg_single}"
print("✓ Single grade calculation works")
```
