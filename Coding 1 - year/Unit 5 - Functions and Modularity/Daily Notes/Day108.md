# Day 108: Functions with Loops

**Learning Target:** I can write functions that use loops to process data repeatedly and produce results.

### Key Concepts

Loops and functions work beautifully together. Inside a function, you often loop through a list, perform calculations, or build new collections. **Combining loops with functions** lets you create powerful data-processing tools.

Common patterns:
- **Accumulation** — Loop through data and build up a result (e.g., sum, count, concatenate)
- **Search** — Loop through data looking for something specific
- **Transformation** — Loop and create a new modified version
- **Aggregation** — Loop and collect statistics (max, min, average)

The key is thinking about what you want to accumulate or collect, and using the loop to build it up.

### Code Examples

```python
# Example 1: Accumulating a sum using a loop
def sum_all(numbers):
    """Return the sum of all numbers in a list."""
    total = 0
    for num in numbers:
        total = total + num
    return total

print(sum_all([1, 2, 3, 4, 5]))  # 15
```

```python
# Example 2: Finding the maximum using a loop
def find_max(numbers):
    """Return the largest number in a list."""
    max_value = numbers[0]
    for num in numbers:
        if num > max_value:
            max_value = num
    return max_value

print(find_max([3, 7, 2, 9, 1]))  # 9
```

```python
# Example 3: Building a new list using a loop
def multiply_by_two(numbers):
    """Return a new list with each number multiplied by 2."""
    result = []
    for num in numbers:
        result.append(num * 2)
    return result

print(multiply_by_two([1, 2, 3]))  # [2, 4, 6]
```

```python
# Example 4: Complex loop—counting occurrences
def count_letter(text, letter):
    """Count how many times a letter appears in text."""
    count = 0
    for char in text:
        if char.lower() == letter.lower():
            count = count + 1
    return count

print(count_letter("Hello", "l"))  # 2
```

```python
# Problem 1 Example
def count_zeros(numbers):
    """Count how many zeros are in a list."""
    count = 0
    for num in numbers:
        if num == 0:
            count = count + 1
    return count

print(count_zeros([1, 0, 2, 0, 0, 3]))  # 3
```

```python
# Problem 2 Example
def average(numbers):
    """Calculate the average of a list of numbers."""
    if len(numbers) == 0:
        return 0
    total = 0
    for num in numbers:
        total = total + num
    return total / len(numbers)

print(average([10, 20, 30]))  # 20.0
print(average([85, 90, 78, 92]))  # 86.25
```

```python
# Problem 3 Example
def find_all_matching(items, target):
    """Return a list of all items that match the target."""
    matches = []
    for item in items:
        if item == target:
            matches.append(item)
    return matches

print(find_all_matching([1, 2, 2, 3, 2], 2))  # [2, 2, 2]
print(find_all_matching(["a", "b", "a", "c"], "a"))  # ['a', 'a']
```
