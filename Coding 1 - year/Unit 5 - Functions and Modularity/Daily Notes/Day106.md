# Day 106: Functions with Lists — Passing Lists as Arguments

**Learning Target:** I can write functions that accept lists as parameters and process list data.

### Key Concepts

Lists are collections of data. When you pass a list to a function, you're giving that function access to all the data in the list. This is very useful for writing functions that process collections of data.

Here's an important detail: lists are passed by **reference**, not by value. This means if your function modifies the list, the original list outside the function is also modified. This is different from numbers and strings (which are passed by value). Understand this to avoid unexpected bugs.

**Example:**
```python
def add_item(my_list, item):
    my_list.append(item)  # This modifies the original list!

numbers = [1, 2, 3]
add_item(numbers, 4)
print(numbers)  # [1, 2, 3, 4] — the original was modified!
```

Common operations with lists in functions:
- **Access elements** — `my_list[0]`, `my_list[-1]`
- **Length** — `len(my_list)`
- **Modify** — `my_list.append()`, `my_list.pop()`, `my_list[0] = new_value`
- **Iterate** — `for item in my_list:`
- **Slice** — `my_list[1:3]`

### Code Examples

```python
# Example 1: Function that processes a list
def print_list(items):
    """Print each item in a list."""
    for item in items:
        print(item)

names = ["Alice", "Bob", "Charlie"]
print_list(names)
```

```python
# Example 2: Function that modifies a list (passed by reference)
def add_to_list(items, new_item):
    """Add an item to the list."""
    items.append(new_item)

shopping = ["milk", "bread"]
add_to_list(shopping, "eggs")
print(shopping)  # ['milk', 'bread', 'eggs'] — modified!
```

```python
# Example 3: Function that finds something in a list
def find_longest(words):
    """Find the longest word in a list."""
    longest = ""
    for word in words:
        if len(word) > len(longest):
            longest = word
    return longest

titles = ["Python", "Programming", "Guide"]
result = find_longest(titles)
print(result)  # Programming
```

```python
# Problem 1 Example
def count_items(items):
    """Returns the number of items in a list."""
    return len(items)

print(count_items([1, 2, 3]))  # 3
print(count_items([]))  # 0
print(count_items(["a", "b", "c", "d"]))  # 4
```

```python
# Problem 2 Example
def get_first_and_last(items):
    """Returns first and last items from a list."""
    if len(items) == 0:
        return None  # or raise an error
    return [items[0], items[-1]]

print(get_first_and_last([1, 2, 3, 4, 5]))  # [1, 5]
print(get_first_and_last(["a", "b"]))  # ['a', 'b']
```

```python
# Problem 3 Example
def get_valid_scores(scores):
    """Returns scores of 70 or higher (creates new list)."""
    valid = []
    for score in scores:
        if score >= 70:
            valid.append(score)
    return valid

original = [85, 92, 68, 75, 60, 88]
passing = get_valid_scores(original)
print(f"Original: {original}")  # [85, 92, 68, 75, 60, 88]
print(f"Passing: {passing}")    # [85, 92, 75, 88]
```
