# Day 103: Functions Calling Functions — Composition

**Learning Target:** I can write functions that call other functions to break complex problems into manageable pieces.

### Key Concepts

So far, you've written functions that work independently. But functions become truly powerful when they call *other functions*. This is called **function composition**—building larger functions by combining smaller ones.

Function composition lets you solve big problems by breaking them into smaller, simpler pieces. You write functions for the small pieces, then combine them to solve the big problem. This makes code easier to understand, test, and maintain.

**Example:** To calculate the total cost of a purchase with tax and discount:
1. Write a function to calculate the subtotal
2. Write a function to apply a discount
3. Write a function to apply tax
4. Write a main function that calls all three in order

Each function does one job well. The main function orchestrates them.

When one function calls another, the called function must be defined *before* it's called, or defined anywhere before the caller is called. Python reads the file top to bottom, so helper functions usually go at the top, and the main logic goes at the bottom.

### Code Examples

```python
# Example 1: Simple composition
def square(n):
    return n * n

def sum_of_squares(a, b):
    return square(a) + square(b)

result = sum_of_squares(3, 4)
print(result)  # 9 + 16 = 25
```

```python
# Example 2: Building a calculator with composed functions
def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

def calculate_payment(hours, rate, num_jobs):
    # Compose smaller functions
    pay_per_job = multiply(hours, rate)
    total_pay = multiply(pay_per_job, num_jobs)
    return total_pay

print(calculate_payment(8, 15, 3))  # 8 * 15 * 3 = 360
```

```python
# Example 3: Data processing pipeline
def celsius_to_fahrenheit(celsius):
    return (celsius * 9/5) + 32

def round_temperature(temp):
    return round(temp, 1)

def format_temperature(fahrenheit):
    return f"{fahrenheit}°F"

def convert_and_format(celsius):
    fahrenheit = celsius_to_fahrenheit(celsius)
    rounded = round_temperature(fahrenheit)
    formatted = format_temperature(rounded)
    return formatted

print(convert_and_format(0))    # 32.0°F
print(convert_and_format(100))  # 212.0°F
```

```python
# Problem 1 Example
def celsius_to_fahrenheit(c):
    return (c * 9/5) + 32

def round_temp(f):
    return round(f, 1)

def convert_and_round(celsius):
    fahrenheit = celsius_to_fahrenheit(celsius)
    return round_temp(fahrenheit)

print(convert_and_round(0))    # 32.0
print(convert_and_round(25))   # 77.0
print(convert_and_round(100))  # 212.0
```

```python
# Problem 2 Example
def double(n):
    return n * 2

def add(a, b):
    return a + b

def square(n):
    return n * n

def complex_calculation(x, y):
    x_squared = square(x)
    y_doubled = double(y)
    return add(x_squared, y_doubled)

print(complex_calculation(3, 4))  # 9 + 8 = 17
```

```python
# Problem 3 Example
def get_first_word(text):
    return text.split()[0]

def get_last_word(text):
    return text.split()[-1]

def combine(word1, word2):
    return f"{word1} {word2}"

def swap_words(phrase):
    first = get_first_word(phrase)
    last = get_last_word(phrase)
    return combine(last, first)

print(swap_words("Hello World"))  # World Hello
```
