# Day 107: Functions That Return Lists

**Learning Target:** I can write functions that build and return new lists as their result.

### Key Concepts

Just as functions can return numbers and strings, they can also **return lists**. This is powerful for **data transformation**—taking input data and producing a new collection that's been processed in some way.

When a function returns a list, the caller gets that list and can use it in their program. Common patterns:
- **Filtering** — Create a new list with only certain items (e.g., only positive numbers)
- **Transforming** — Create a new list where each item has been modified (e.g., all doubled)
- **Combining** — Create a new list by merging multiple lists
- **Sorting and organizing** — Return a modified version of input

A key advantage of returning lists (rather than modifying the original) is that the original data stays intact. Your function becomes **pure** and predictable—it takes input and produces output without side effects.

### Code Examples

```python
# Example 1: Filtering—return only items that meet a condition
def get_even_numbers(numbers):
    """Return a list of only even numbers."""
    evens = []
    for num in numbers:
        if num % 2 == 0:
            evens.append(num)
    return evens

result = get_even_numbers([1, 2, 3, 4, 5, 6])
print(result)  # [2, 4, 6]
```

```python
# Example 2: Transforming—apply a change to each item
def double_all(numbers):
    """Return a new list with each number doubled."""
    doubled = []
    for num in numbers:
        doubled.append(num * 2)
    return doubled

result = double_all([1, 2, 3])
print(result)  # [2, 4, 6]
```

```python
# Example 3: Filtering and transforming together
def get_long_names(names):
    """Return only names with 5+ characters, in uppercase."""
    result = []
    for name in names:
        if len(name) >= 5:
            result.append(name.upper())
    return result

result = get_long_names(["Al", "Grace", "Bob", "Charlie"])
print(result)  # ['GRACE', 'CHARLIE']
```

```python
# Problem 1 Example
def make_uppercase(words):
    """Return a list of words converted to uppercase."""
    result = []
    for word in words:
        result.append(word.upper())
    return result

original = ["hello", "world", "python"]
result = make_uppercase(original)
print(f"Original: {original}")  # ['hello', 'world', 'python']
print(f"Result: {result}")      # ['HELLO', 'WORLD', 'PYTHON']
```

```python
# Problem 2 Example
def filter_by_length(words, min_length):
    """Return only words with at least min_length characters."""
    result = []
    for word in words:
        if len(word) >= min_length:
            result.append(word)
    return result

words = ["the", "quick", "brown", "fox", "jumps"]
print(filter_by_length(words, 4))  # ['quick', 'brown', 'jumps']
```

```python
# Problem 3 Example
def remove_duplicates(items):
    """Return a list with duplicates removed, preserving order."""
    seen = []
    result = []
    for item in items:
        if item not in seen:
            result.append(item)
            seen.append(item)
    return result

data = [1, 2, 2, 3, 1, 4, 3, 5]
print(remove_duplicates(data))  # [1, 2, 3, 4, 5]
```
