# Day 109: Functions with Conditionals

**Learning Target:** I can write functions that use conditional statements to make decisions based on input data.

### Key Concepts

Functions often need to **make decisions**. Inside a function, `if`, `elif`, and `else` statements let your function behave differently depending on what it receives.

**Decision-making functions** are especially useful because they encapsulate logic—the caller doesn't need to know how the decision is made, just what the result is. For example, a function that checks if a password is valid can hide the validation rules inside, and the caller just gets back `True` or `False`.

Common patterns:
- **Validation** — Check if input meets requirements; return `True` or `False`
- **Classification** — Examine input and return a category or type
- **Conditional processing** — Process input differently based on conditions
- **Error handling** — Return an error or default value if something's wrong

### Code Examples

```python
# Example 1: Validation function
def is_adult(age):
    """Return True if age >= 18, False otherwise."""
    if age >= 18:
        return True
    else:
        return False

print(is_adult(17))  # False
print(is_adult(21))  # True
```

```python
# Example 2: Classification function
def grade_level(age):
    """Return the likely grade level based on age."""
    if age < 6:
        return "Preschool"
    elif age <= 7:
        return "1st Grade"
    elif age <= 8:
        return "2nd Grade"
    elif age <= 12:
        return "Middle School"
    else:
        return "High School"

print(grade_level(7))  # 1st Grade
print(grade_level(14))  # High School
```

```python
# Example 3: Conditional processing
def get_discount(purchase_amount):
    """Return the discount percentage based on purchase amount."""
    if purchase_amount >= 100:
        return 0.20  # 20% discount
    elif purchase_amount >= 50:
        return 0.10  # 10% discount
    elif purchase_amount >= 20:
        return 0.05  # 5% discount
    else:
        return 0  # No discount

amount = 75
discount = get_discount(amount)
final = amount * (1 - discount)
print(f"Final price: ${final}")
```

```python
# Example 4: Validation with detailed feedback
def is_strong_password(password):
    """Check if password is strong (8+ chars, has number and letter)."""
    if len(password) < 8:
        return False
    has_letter = False
    has_number = False
    for char in password:
        if char.isalpha():
            has_letter = True
        if char.isdigit():
            has_number = True
    return has_letter and has_number

print(is_strong_password("abc"))  # False
print(is_strong_password("password"))  # False
print(is_strong_password("pass1234"))  # True
```

```python
# Problem 3 Example
def validate_email(email):
    """Check if email has valid format (contains @ and .)."""
    if email.count("@") != 1:
        return False
    at_position = email.index("@")
    after_at = email[at_position + 1:]
    if "." not in after_at:
        return False
    return True

print(validate_email("alice@example.com"))  # True
print(validate_email("bob@example"))        # False
print(validate_email("charlie@@domain.com")) # False
```
