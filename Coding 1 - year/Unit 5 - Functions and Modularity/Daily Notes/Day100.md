# Day 100: Practice Lab — Building a Function Library

**Learning Target:** I can design and build a library of related functions and test them together in a complete program.

### Overview

This lab brings together everything from Week 20. You'll create your own **function library**—a set of related functions that work together. Your library will have a clear theme (e.g., math utilities, string tools, game helpers, or student graders) and each function should have a specific job.

### Requirements

Choose a theme and create a library of **5–8 functions**. Each function should:
- Have a clear, descriptive name
- Accept parameters (at least one)
- Return a value (not just print)
- Be tested thoroughly

### Suggested Themes

1. **Math Utilities:** Functions like `is_even()`, `gcd()`, `factorial()`, `is_prime()`, `absolute_value()`
2. **String Tools:** Functions like `reverse_string()`, `count_vowels()`, `is_palindrome()`, `capitalize_words()`
3. **Game Helpers:** Functions like `roll_dice()`, `is_valid_move()`, `calculate_score()`, `get_random_question()`
4. **Grade Calculator:** Functions like `letter_grade()`, `average()`, `weighted_score()`, `is_passing()`
5. **Temperature Converter:** Functions like `celsius_to_fahrenheit()`, `fahrenheit_to_celsius()`, `is_freezing()`, `feels_hot()`

### Example: Math Utilities Library

```python
# Math Utilities Library

def is_even(n):
    """Returns True if n is even, False otherwise."""
    return n % 2 == 0

def is_odd(n):
    """Returns True if n is odd, False otherwise."""
    return n % 2 != 0

def absolute_value(n):
    """Returns the absolute value of n."""
    if n < 0:
        return -n
    else:
        return n

def power(base, exponent):
    """Returns base raised to the exponent power."""
    result = 1
    for i in range(exponent):
        result = result * base
    return result

def is_perfect_square(n):
    """Returns True if n is a perfect square."""
    root = int(n ** 0.5)
    return root * root == n

# Test the library
if __name__ == "__main__":
    print("=== Math Utilities Library ===")

    print(f"is_even(4): {is_even(4)}")      # True
    print(f"is_even(7): {is_even(7)}")      # False

    print(f"is_odd(4): {is_odd(4)}")        # False
    print(f"is_odd(7): {is_odd(7)}")        # True

    print(f"absolute_value(-5): {absolute_value(-5)}")  # 5
    print(f"absolute_value(3): {absolute_value(3)}")    # 3

    print(f"power(2, 3): {power(2, 3)}")    # 8
    print(f"power(5, 2): {power(5, 2)}")    # 25

    print(f"is_perfect_square(16): {is_perfect_square(16)}")  # True
    print(f"is_perfect_square(15): {is_perfect_square(15)}")  # False
```
