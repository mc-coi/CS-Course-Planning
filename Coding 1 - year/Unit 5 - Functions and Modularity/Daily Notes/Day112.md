# Day 112: The main() Function Pattern

**Learning Target:** I can organize my program using the main() function pattern to create clean, modular code.

### Key Concepts

A common and professional pattern in Python is to write a **`main()` function** that orchestrates your entire program. The `main()` function:
- Calls other functions in the right order
- Controls the overall program flow
- Is called once, at the very bottom of the file

This pattern keeps your program clean and organized. It's especially useful because:
- **Clarity** — You can see the whole program at a glance by reading `main()`
- **Modularity** — All the helper functions are separate and reusable
- **Testability** — You can test individual functions without running the whole program
- **Convention** — Professional Python code uses this pattern

The typical structure is:
```python
# Helper functions first
def function_1():
    ...

def function_2():
    ...

# Main function near the end
def main():
    function_1()
    function_2()
    ...

# Call main at the very bottom
if __name__ == "__main__":
    main()
```

The **`if __name__ == "__main__":`** line is a special Python idiom. It means "only run this code if this file is being run directly (not imported)". This is a Python best practice.

### Code Examples

```python
# Example 1: Simple main() pattern
def greet_user():
    name = input("What's your name? ")
    print(f"Hello, {name}!")

def show_menu():
    print("1. Say hello")
    print("2. Goodbye")

def main():
    """Main program flow."""
    show_menu()
    greet_user()

if __name__ == "__main__":
    main()
```

```python
# Example 2: More complex program with main()
def get_numbers():
    """Get two numbers from user."""
    a = float(input("First number: "))
    b = float(input("Second number: "))
    return a, b

def show_menu():
    """Display operation choices."""
    print("\n=== Calculator ===")
    print("1. Add")
    print("2. Subtract")
    print("3. Multiply")

def calculate(a, b, operation):
    """Perform calculation."""
    if operation == "1":
        return a + b
    elif operation == "2":
        return a - b
    elif operation == "3":
        return a * b

def display_result(a, b, result, operation):
    """Show the result."""
    print(f"\n{a} and {b}: Result = {result}")

def main():
    """Main program."""
    show_menu()
    a, b = get_numbers()
    op = input("Choose operation (1-3): ")
    result = calculate(a, b, op)
    display_result(a, b, result, op)

if __name__ == "__main__":
    main()
```

```python
# Example 3: Game with main() pattern
def load_game():
    """Initialize game state."""
    return {"score": 0, "lives": 3}

def play_round(game_state):
    """Play one game round."""
    # Game logic here
    game_state["score"] += 10
    return game_state

def display_score(game_state):
    """Show current score."""
    print(f"Score: {game_state['score']}")

def is_game_over(game_state):
    """Check if game should end."""
    return game_state["lives"] <= 0

def main():
    """Main game loop."""
    game = load_game()

    while not is_game_over(game):
        game = play_round(game)
        display_score(game)

        play_again = input("Play again? (y/n): ")
        if play_again.lower() != "y":
            break

    print(f"Game over! Final score: {game['score']}")

if __name__ == "__main__":
    main()
```

# Problem 2 Example (Grade Manager reorganized)
def get_grades():
    grades = []
    while True:
        try:
            grade = float(input("Enter grade (or 'done'): "))
            grades.append(grade)
        except:
            break
    return grades

def calculate_average(grades):
    if not grades:
        return 0
    return sum(grades) / len(grades)

def get_letter_grade(avg):
    if avg >= 90: return 'A'
    elif avg >= 80: return 'B'
    elif avg >= 70: return 'C'
    else: return 'F'

def display_report(grades):
    avg = calculate_average(grades)
    letter = get_letter_grade(avg)
    print(f"\nAverage: {avg:.2f}")
    print(f"Grade: {letter}")

def main():
    grades = get_grades()
    display_report(grades)

if __name__ == "__main__":
    main()
