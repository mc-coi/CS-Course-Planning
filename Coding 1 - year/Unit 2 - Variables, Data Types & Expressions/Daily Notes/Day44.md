### Assessment Overview

The Unit 2 Assessment evaluates your understanding of variables, data types, expressions, and your ability to build complete programs. The assessment has multiple sections designed to measure different skills.

### Assessment Format

**Total Points: 100**

**Section 1: Multiple Choice (20 points, 1 point each)**
- Identify data types
- Predict output
- Understand operators
- Recognize errors

**Section 2: Short Answer (20 points, 5 points each)**
- Explain concepts
- Describe processes
- Identify best practices

**Section 3: Code Tracing (20 points, 4 points each)**
- Predict program output
- Identify variable values
- Trace through calculations

**Section 4: Code Writing (40 points)**
- Write programs to solve problems
- Demonstrate proper syntax
- Show correct logic

### What to Study

**Definitely know:**
- [ ] Data types: int, float, str, bool
- [ ] Variable assignment and reassignment
- [ ] Type conversion (int(), float(), str())
- [ ] String indexing [i] and slicing [start:stop:step]
- [ ] String methods (.upper(), .lower(), .strip(), .replace(), .find(), .count())
- [ ] Arithmetic operators and PEMDAS
- [ ] Comparison operators (==, !=, <, >, <=, >=)
- [ ] F-string formatting {value:.2f}
- [ ] Augmented assignment (+=, -=, etc.)
- [ ] Multiple assignment and swapping
- [ ] Math module (math.sqrt(), math.pi)
- [ ] Building complete programs with input/output

**Important edge cases:**
- [ ] Division `/` returns float (use `//` for integer division)
- [ ] Strings are immutable (can't change individual characters)
- [ ] Indexing starts at 0
- [ ] Slice stop is exclusive
- [ ] Negative indices count from the end
- [ ] `input()` returns a string (need type conversion)
- [ ] `input().strip()` removes whitespace

### Sample Assessment Questions

**Multiple Choice Example:**
What is the output of `print("Python"[1:4])`?
a) `"yth"`
b) `"ytho"`
c) `"tho"`
d) `"yt"`

**Short Answer Example:**
Explain the difference between `=` and `==` and give an example of how each is used.

**Code Trace Example:**
```python
x = 10
y = 3
x = x + y
y = x - y
x = x - y
print(x, y)
```
What does this print?

**Code Writing Example:**
Write a program that:
- Asks for an item price
- Asks for a tip percentage
- Calculates tip amount
- Displays formatted total with tip

### Test-Taking Strategies

**Before you start:**
1. Read all questions carefully
2. Manage your time (roughly 20 min per section)
3. Answer easy questions first, hard ones later
4. Review your answers if time permits

**For multiple choice:**
1. Read all options before answering
2. Eliminate obviously wrong answers
3. Choose the BEST answer if uncertain

**For code tracing:**
1. Work through step-by-step
2. Track variable values carefully
3. Pay attention to operators (/, //, %, etc.)

**For code writing:**
1. Plan before coding (what inputs, calculations, outputs?)
2. Use clear variable names (snake_case)
3. Test your code mentally (does it work?)
4. Add comments if time allows
