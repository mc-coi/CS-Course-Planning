## Common Mistakes Addressed

The course explicitly teaches students to avoid:
1. Using `=` instead of `==`
2. Forgetting colons after conditions
3. Incorrect indentation
4. Using `and` instead of `or` (and vice versa)
5. Wrong order in elif chains (general before specific)
6. Off-by-one errors (> vs >=)
7. Not validating input
8. Type mismatches in comparisons
9. Impossible logic conditions
10. Skipping necessary conditional branches
