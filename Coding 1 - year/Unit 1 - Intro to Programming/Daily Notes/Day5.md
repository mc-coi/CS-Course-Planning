### Key Concepts
This day is for hands-on practice and consolidation. You'll apply everything from Week 1:

A **practice lab** is hands-on time to build confidence with the fundamentals before moving forward. You'll create your own program from scratch with creative freedom, combining multiple concepts.

### Guided Practice
Create one of the following projects (your choice):

**Option A: ASCII Art Gallery**
- Create a program that prints 3 different ASCII art images
- Each should be labeled with a comment explaining what it is
- Use formatting (\n, spacing) to make it visually interesting
- Include a title for the whole program

**Option B: Formatted Information Card**
- Create a "card" (like a business card or sports card)
- Include formatted sections with borders using special characters
- Use tabs and newlines to align everything nicely
- Include your information or a fictional character's information

**Option C: ASCII Animation (Static)**
- Create a program that displays multiple "frames" of ASCII art
- You could show a rocket launching, a sun rising, or a stick figure moving
- Use comments to explain each frame
- Include proper spacing between frames

### Practice Problems

**Problem 1 — Beginner:** Create a program that prints a simple shape (rectangle, triangle, or diamond) using * characters.

**Problem 2 — Intermediate:** Create a formatted "information card" for a character from a book, movie, or game. Include name, age, special powers, and a border made of special characters.

**Problem 3 — Challenge:** Create a program that prints a more complex ASCII art image (at least 8 lines) with a title and description, properly formatted and commented.
