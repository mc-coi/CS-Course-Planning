##  What's Here

You'll find a **complete course curriculum** in the **Coding I** and **Coding II** folders above — built from scratch, fully aligned to Tennessee state CS standards, and ready to use on day one.

Every unit includes daily lesson plans, student-facing notes, practice problem banks, reference guides, and unit assessments — in both editable Word (.docx) and printable PDF formats.

### Each hyperbook has:
- **Day-by-day lesson notes** with key concepts and code examples
- **Practice problems** at Beginner, Intermediate, and Challenge levels — with hints and answers
- **Reference guides** with syntax cheatsheets and common mistake tables
- **Unit assessments** with review questions and answer keys
- **Built-in search** — press `/` to find any topic instantly
- **Keyboard navigation** — use `←` `→` arrow keys to flip between pages

```

📁 Coding II/
│
├── 📗 Unit 1 — Data Structures
├── 📗 Unit 2 — Object-Oriented Programming
├── 📗 Unit 3 — Advanced Functions & Algorithms
├── 📗 Unit 4 — File I/O & Data Processing
```

Each unit folder contains:

```
Unit N - Topic Name/
├── 📄 Unit N - Topic.md          ← Complete unit overview (all days)
├── 📄 UnitAssessment.md          ← 10 review Q&As
├── 📄 UnitPractice.md            ← 15 practice problems
├── 📄 ReferenceGuide.md          ← Key syntax & vocab
├── 📁 Daily Notes/
│   ├── Day1.md … DayN.md         ← One file per class day
├── 📁 Daily Lesson Plans/
│   ├── 📁 Word/                  ← Editable .docx lesson plans
│   └── 📁 PDF/                   ← Printable .pdf lesson plans
```
