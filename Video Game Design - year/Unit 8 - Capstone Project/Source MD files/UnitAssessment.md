## Grading Scale
- **A (90–100):** Exceptional. Game is fully polished, fun, and complete. You should be proud of this.
- **B (80–89):** Very good. Game is fun and mostly polished. Minor areas for improvement.
- **C (70–79):** Good. Game is playable and meets requirements. Some rough edges.
- **D (60–69):** Meets minimum standards. Game works but needs more polish or refinement.
- **F (Below 60):** Does not meet minimum standards. Game is incomplete or broken.

## Teacher Scoring Guide

### How to Use This Rubric

1. **Play the game fully** from start to finish before scoring.
2. **Test for bugs:** Try to break it. Try edge cases.
3. **Score each category** using the point scale (4 = Excellent, 3 = Proficient, etc.).
4. **Note observations:** Jot down specific examples from the game to justify scores.
5. **Add comments:** Tell the student what they did well and where they can improve.

### Key Observations to Look For

**Technical (40 points):**
- Does the player control feel responsive? Are there input delays?
- Are enemies/obstacles challenging but fair?
- Do all scenes load correctly? Any crashes?
- Is there a clear win state and lose state?
- Do physics and collisions work as intended?
- Is audio present and does it enhance gameplay?
- Is the UI professional and easy to use?
- Is the code clean and organized?
- Did the game run smoothly without lag?

**Design (28 points):**
- Is the core mechanic clear and fun?
- Does difficulty scale appropriately?
- Do all visuals fit a cohesive theme?
- Is audio intentional and polished?
- Does the game feel original or is it derivative?
- Is the game content-complete?
- Is it fun for the target audience?

**Polish (20 points):**
- Does the game *feel* good to play? (Movement, feedback, animations)
- Is the menu professional and intuitive?
- Are there visual glitches or stretching?
- Does the GDD explain the design well?
- Does the exported build work?

**Documentation (16 points):**
- Is the GDD complete and professional?
- Is the code commented?
- Is there a development log or journal?
- Are presentation materials included?
