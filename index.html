What's Included

What you'll find
in each course.

Every unit follows the same structure, so you always know where to look. Use what works, skip what doesn't.

📅

Daily Lesson Notes

Student-facing markdown notes for every single class day — key concepts, code examples, guided practice, and problems with hints and answers.

📋

Lesson Plan Docs

Structured lesson plans in both editable Word (.docx) and printable PDF format for every day of the semester.

💪

Practice Problem Banks

15 problems per unit across three tiers — Beginner, Intermediate, and Challenge — so every student is appropriately stretched.

📖

Reference Guides

Unit-level syntax cheatsheets, vocabulary lists, and common mistake tables students can use while coding or studying for tests.

📝

Unit Assessments

10-question assessments per unit with full answer keys, plus detailed unit assessment prep Q&As for student review.

🔍

Interactive Hyperbooks

All content compiled into searchable single-page HTML books with syntax highlighting, keyboard nav, and reading progress tracking.

About

Hey, I'm Zach.

I'm a high school CS teacher in Tennessee. I built these materials for my own classes and figured they might save someone else some time too.

Everything here is free and open — aligned to Tennessee 9–12 CS standards, but flexible enough to adapt to your classroom. Use it as-is, remix it, or just grab the parts that help.

If you have questions or want to share ideas, I'd love to hear from you.
