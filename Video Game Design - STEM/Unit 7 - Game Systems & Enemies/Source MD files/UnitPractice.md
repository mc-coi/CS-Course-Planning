# Unit Practice Bank

1. **Beginner:** Modular Health System - Create a Health component and attach it to 3+ different entity types (player, enemy, destructible).

2. **Beginner:** Damage Types - Implement different damage types (sword, fire, poison) that scale differently based on armor/resistance.

3. **Beginner:** Knockback Physics - When hit, push the entity away from the hit source. Implement knockback on damage.

4. **Beginner:** Invulnerability Frames - Implement damage immunity after taking damage (visual blink effect + frame counter).

5. **Intermediate:** Enemy Patrol AI - Create a patrolling enemy with adjustable patrol points and speed.

6. **Intermediate:** Enemy Detection - Use raycasting to detect line-of-sight to the player. Only chase if visible.

7. **Intermediate:** NavMesh Pathfinding - Use Unity's NavMesh system for enemy pathfinding. Compare to manual movement.

8. **Intermediate:** Attack Cooldown - Implement cooldown between enemy attacks. Show cooldown timer visually.

9. **Challenge:** Weakness System - Implement weaknesses (enemy weak to fire, strong to ice). Damage varies by element.

10. **Challenge:** Boss Enemy - Create a boss enemy with multiple phases and special attacks.

11. **Challenge:** Loot Drop - When enemy dies, randomly drop loot (health, coins, items).

12. **Challenge:** Difficulty Scaling - Create difficulty settings that change enemy health, speed, and damage.

13. **Challenge:** Wave System - Create waves of enemies with increasing difficulty. Track which wave the player is on.

14. **Challenge:** Time-Based Win Condition - Implement a survival mode where player wins by lasting X minutes.

15. **Challenge:** Game State Flowchart - Create a complete flowchart of your game's state transitions and document each state's behavior.
