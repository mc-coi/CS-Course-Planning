# Unit Practice Bank

1. **Beginner - Hierarchy Organization:** Create a scene with 10 GameObjects. Organize them into a logical parent-child hierarchy (e.g., a "Player" parent with "Head," "Body," "Legs" children). Screenshot the Hierarchy panel.

2. **Beginner - Transform Precision:** Use the Inspector to position a Cube at exactly (3.5, 2.0, -1.5). Verify using the Move tool that you can position it to the exact coordinates.

3. **Beginner - Material Blending:** Create two materials with different colors. Assign one to a Cube and another to a Sphere. Adjust their Smoothness and Metallic values and describe the visual differences.

4. **Beginner - Lighting Experiment:** Place a Cube in a dark scene (no lights). Add a Point Light near it. Move the light around and observe how the lighting changes. What happens when you increase the light's intensity?

5. **Intermediate - Script Attachment:** Write a script that prints the time elapsed since the scene started. Attach it to a Cube and run the scene. What does the Console show?

6. **Intermediate - Debug Logging:** Write a script that logs the frame count every 10 frames. Attach it to any GameObject and observe the pattern in the Console.

7. **Intermediate - Public vs. Private:** Create a script with both public and private variables. In the Inspector, verify that you can only edit the public ones.

8. **Intermediate - GetComponent:** Write a script that gets the Transform component of another GameObject in the scene (using GetComponent) and logs its position.

9. **Intermediate - Scale Manipulation:** Using the Scale tool (R key), make a Cube 2x larger in the X and Z axes, but 0.5x smaller in the Y axis. What's the resulting shape?

10. **Intermediate - Scene Navigation:** Create a scene with 20 GameObjects. Use the Hierarchy's search function to find a specific GameObject by name.

11. **Challenge - Viewport Control:** In the Scene view, use middle-mouse drag to pan, scroll wheel to zoom, and right-click drag to rotate. Navigate to view your scene from 5 different angles.

12. **Challenge - Time Analysis:** Write a script that prints `Time.deltaTime` to the Console every frame. What range of values do you see? Why does this matter?

13. **Challenge - Color Interpolation:** Write a script that smoothly changes a Cube's color from red to blue over 5 seconds.

14. **Challenge - Rotation Display:** Write a script that prints the current rotation (in degrees) every second. Rotate the GameObject and see the values change.

15. **Challenge - Audio Cue:** Design a game event where collecting an object plays a sound and changes the object's color. Write pseudocode (plain English) describing the steps you'd take.
